# Requirements

* Rails 7
* Bundler
# How to install

* `git clone git@github.com:spaquet/rails-flash-msg.git`
* `cd rails-flash-msg`
* `bundle install`
* `rails db:create`
* `rails db:migrate`
* `./bin/dev` to start the project.
